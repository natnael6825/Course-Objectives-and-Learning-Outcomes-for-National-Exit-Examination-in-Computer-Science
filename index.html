Course Objectives and Learning Outcomes for National Exit Examination Bachelor of Science Degree in Computer Science


Course Name General Objective Learning Outcomes
Software Engineering Identifying a problem, studying the problem, gathering data, and relevant materials and an open presentation of the development of the project work Understand software analysis, design, and development techniques and tools; Demonstrate conceptual and technical skills in the analysis, design, and implementation of a software system; Create requirements using use case modeling concepts
Web Programming Develop the ability to plan, develop, test, and debug web pages logically. Understand concepts, principles, and methods in programming for web and Internet environment; Apply Server Side scripting Languages and Implement client-side interactivity; Specify, build and manage form and content of information-rich web sites; Design, implement and evaluate client-server systems following specific protocol specifications, taking into account concurrency issue
Fundamental of Database Systems Introduce database management systems, with a focus on how to organize, maintain, and retrieve data efficiently and effectively Understand the principles of database design; Apply the database concepts to real-world database design; Design database systems for real-world scenarios
Computer Programming Introduce a disciplined approach to problem-solving methods and algorithm development, as well as to teach the syntax and vocabulary of a modern programming language Understand the principles of computer programming; Design, code, debug and solve problems using programming concepts; Apply the principle of mathematical knowledge to prove statements and solve problems in computing science
Object Oriented Programming Demonstrate how real-world entities such as inheritance, hiding, polymorphism, and son can be implemented in programming Understand the basic object-oriented concepts; Successfully code, debug and run programs with object-oriented principles; Apply object-oriented programming concepts to solve problems
Design and Analysis of Algorithms Understanding of algorithm design and analysis for a variety of problems, as well as developing skills to reason about and prove algorithm properties Understand algorithm design and analysis principles; Apply algorithmic concepts to solve computing problems; Evaluate the efficiency and effectiveness of algorithms
Data Structure and Algorithms Introduce data structures and algorithms concepts Understand data structure concepts; Apply data structures to solve computing problems; Analyze the performance of data structures and algorithms
Data Communication and Computer Networking Introduce fundamental concepts of data communication and computer networking Understand data communication concepts; Apply networking principles to design and implement network systems; Analyze network protocols and architecture
Computer Security Provide an understanding of computer security concepts, techniques, and applications Understand computer security principles; Apply security techniques to protect computing systems; Analyze security threats and vulnerabilities
Network and System Administration Introduce network and system administration concepts Understand network and system administration principles; Apply administration techniques to manage network systems; Evaluate the performance and security of network systems
Introduction to Artificial Intelligence Introduce artificial intelligence concepts and techniques Understand artificial intelligence concepts; Apply AI techniques to solve real-world problems; Evaluate AI systems and their impact
Operating System Introduce operating system concepts and functions Understand operating system principles; Apply OS concepts to manage computing resources; Analyze OS performance and security
Computer Organization and Architecture Introduce computer organization and architecture concepts Understand computer organization principles; Apply architecture concepts to design and analyze computer systems; Evaluate the performance of computer systems
Automata and Complexity Theory Introduce automata and complexity theory concepts Understand automata and complexity theory concepts; Apply theoretical principles to solve computational problems; Evaluate the complexity of algorithms
Compiler Design Introduce compiler design concepts and techniques Understand compiler design principles; Apply techniques to develop and optimize compilers; Evaluate compiler performance
